## 📌 Overview

React E-Commerce Application is a modern and responsive shopping platform built using **React.js**. The application provides a complete online shopping experience with user authentication, product management, cart functionality, wishlist management, and order tracking.

The project follows a **component-based architecture** and focuses on creating a seamless and user-friendly shopping experience while implementing modern React concepts and state management techniques.

## 🚀 Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/likitha-yarraguntla/react-ecommerce-app.git
```

### Navigate to Project Directory

```bash
cd react-ecommerce-app
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
